# History Application - تطبيق التاريخ الطبي

تطبيق بسيط لتسجيل بيانات المرضى والتاريخ الطبي باستخدام Flutter.

## المميزات
- إضافة مرضى جدد
- تسجيل العمر
- تحديد أنواع التاريخ الطبي (Medical History, Dental History, Extra Oral, Intra Oral)
- حفظ البيانات محلياً باستخدام Hive

## كيفية التثبيت
1. تحميل ملف APK من الإصدارات
2. تثبيته على جهاز Android
3. البدء في الاستخدام
